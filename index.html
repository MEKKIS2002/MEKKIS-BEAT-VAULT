💾 Trygg lagring av beats

Beats lastet opp via fil-upload lagres som midlertidige object URLs i nettleseren. Disse forsvinner når du lukker fanen — men tekster, ratings og metadata beholdes i localStorage.

For trygg, varig lagring anbefaler vi:

☁️ Google Drive (anbefalt)
Last opp beats til Drive-mappen, legg inn lenken i "URL / Drive-lenke"-feltet. Beats spilles av direkte fra Drive og overlever alle sesjoner.
🖥️ Lokal server (teknisk)
Kjør en enkel lokal server (f.eks. med Node.js) i mappen der beatsa ligger. Lenk til http://localhost:3000/beat.mp3 — fungerer så lenge serveren er oppe.
📤 Eksporter backup regelmessig
Bruk "Eksporter backup"-knappen øverst for å lagre alt til en JSON-fil. Tekster, ratings og metadata er alltid trygge der.

SoundCloud oEmbed

Hentes direkte uten proxy for offentlige tracks. Private tracks krever proxy.
